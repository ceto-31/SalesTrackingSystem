Drop your notification sound here as: notification.mp3

Recommended:
- Format: MP3 (most compatible) or OGG
- Length: 0.5 - 2 seconds (short chime / ding)
- Size: under 100 KB
- Volume: pre-normalized; the player will play at ~70%

This file is served at /sounds/notification.mp3 by Vite/Vercel.
The admin tab will play it whenever the new-order count goes up.

Note: most browsers block audio until the user interacts with the page
at least once (click anywhere). This is a browser policy, not a bug.
